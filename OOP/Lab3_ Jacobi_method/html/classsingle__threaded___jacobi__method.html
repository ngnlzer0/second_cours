◆ Main_method()

virtual void single_threaded_Jacobi_method::Main_method ( double epsilon)
virtual

Performs the Jacobi iterative algorithm to solve the system.

Algorithm steps:

  1. Checks if matrix A is diagonally dominant.
  2. Initializes X_prev and X_cur with zeroes.
  3. Iteratively updates x[i] using: x_cur[i] = (1.0 / D[i][i]) * (B[i] - SumR(i, x_prev));
  4. Continues until all |x_cur[i] - x_prev[i]| < epsilon.
Parameters
epsilonDesired accuracy
